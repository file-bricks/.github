# file-bricks

**Local-first desktop software for file management, knowledge capture, and everyday information workflows.**

file-bricks builds cross-platform applications for organizing, searching, synchronizing, browsing, and protecting local data. The focus is practical desktop software that works on your machine first, with optional AI support where it adds real value.

## Selected Apps

| App | Description |
|-----|-------------|
| [ExplorerPro](https://github.com/file-bricks/ExplorerPro) | Advanced file explorer with preview panels, privacy monitoring, and sync-aware workflows |
| [ProFiler](https://github.com/file-bricks/ProFiler) | Professional file management suite with full-text search, OCR, and document-heavy workflows |
| [ProSync](https://github.com/file-bricks/ProSync) | Intelligent backup synchronization with database safety and scheduled syncs |
| [AmpelClip](https://github.com/file-bricks/AmpelClip) | Clipboard privacy monitor with automatic anonymization of sensitive data |
| [NoteSpaceLLM](https://github.com/file-bricks/NoteSpaceLLM) | Local, privacy-first alternative to notebook-style document analysis tools |
| [knowledgedigest](https://github.com/file-bricks/knowledgedigest) | Portable knowledge database with local indexing, search, and optional LLM summaries |
| [ProfiPrompt](https://github.com/file-bricks/ProfiPrompt) | Desktop prompt manager for organizing, versioning, and exporting AI prompts |
| [MetaWiki](https://github.com/file-bricks/MetaWiki) | Modular knowledge scaffold with 630+ compact stubs for AI-assisted knowledge work |
| [SoftwareCenter](https://github.com/file-bricks/SoftwareCenter) | Lightweight desktop organizer for software shortcuts and launch surfaces |
| [SQLiteViewer](https://github.com/file-bricks/SQLiteViewer) | Zero-dependency SQLite browser with schema view, SQL editor, and CSV export |
| [WinStorePackager](https://github.com/file-bricks/WinStorePackager) | GUI tool for preparing Python desktop apps for Microsoft Store packaging |
| [RSS-BOOK](https://github.com/file-bricks/RSS-BOOK) | Privacy-first RSS and Atom reader that stores feeds as browser bookmarks |
