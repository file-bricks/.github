# file-bricks .github

Organization profile and default community health files for [file-bricks](https://github.com/file-bricks).

The public organization landing page is maintained in [`profile/README.md`](profile/README.md). The repository also contains shared issue templates, pull request template, security policy, contribution notes, code of conduct, funding metadata, and [`llms.txt`](llms.txt) for machine-readable ecosystem navigation.
